# OHIL-Wrapped-2024
Repository for the Web Development Teams annual wrap up of activity

## Run Locally

Clone the project

```bash
  git clone https://github.com/blackberggroup/OHIL-Wrapped-2024.git
```

Go to the project directory

```bash
  cd ohil-wrapped-2024
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Deployment

Project is deployed to production server upon branch merge to main.
